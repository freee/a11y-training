# freee Accessibility Training

[freee株式会社](https://corp.freee.co.jp/) の従業員向けアクセシビリティー研修で使用する、アクセシビリティーの高い例と低い例を体験するためのページです


## 開発環境

Next.js を使用しています。


```bash
# 依存パッケージのインストール
npm install

# 開発サーバーの起動
npm run dev
```

[http://localhost:3000](http://localhost:3000) にて、ローカルで生成されたページを見ることができます。
